# 查询审核

数据源负责人通过 审核 -> 查询 页面查看需要审核的查询工单(未开启查询审核时工单将自动批准)

# 查询详情

点击对应查询工单详情按钮后进入，可查看该工单的基本信息以及执行的SQL语句。
